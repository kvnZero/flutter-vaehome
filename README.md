## 运行方式

- 运行启动您的应用
```dart
  flutter packages get
  flutter run
```
- 使用包
  - flutter_swiper : ^1.1.6
  - sticky_headers : ^0.1.8
  - photo_view : ^0.9.1
  - badges: ^1.1.0
